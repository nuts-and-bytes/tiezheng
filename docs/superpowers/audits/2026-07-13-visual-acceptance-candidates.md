日期：2026-07-13
环境：`localhost:4173`（vite preview + 本地灌入的 seed 数据）。**未在 tiezheng.pages.dev 上做任何写入。**

这是**指控清单，不是判决**。每条都要经过独立 agent 读代码证伪后，才算数。

## 起因

原本只打算验一件事：L（token 统一）把 Chart.js 的 `defaults.borderColor` 从 iOS 系统灰
`#2c2c2e` 换成了 `--color-line`（rgba 白 .07），淡了一半，怕读不出。

结果第一眼就发现这个担心问错了方向 —— `ChartJS.defaults.borderColor` 对这两个图**根本没生效**：
StatsScreen 的 Line 配置里各自硬写了 `grid: { color: 'rgba(255,255,255,0.05)' }`，把 default 整个盖掉。
真正在画的是这个绕过 token 的值。**L 声称"消灭了第 2 个实现点"，其实没搜到这两处。**

这本身就说明：光靠 grep 色值字面量搜不干净，还得看渲染出来的东西。
